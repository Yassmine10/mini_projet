# Design Builder (test pour un stage)

Application web pour configurer et prévisualiser un design de page (bannière + 4 colonnes).

## Stack

- **Frontend** : Next.js 15 + TypeScript
- **Persistance** : localStorage (navigateur)

## Lancer le projet

```bash
npm install
npm run dev
```

Disponible sur `http://localhost:3000`

> Aucun backend requis. Les données sont sauvegardées automatiquement dans le navigateur.
